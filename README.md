# Medical-Assistant

```
To create the virtual environment use the below command
  uv venv
```

```
To activate the virtual environment use the below command
  source .venv/bin/activate
```

```
To install the requirements.txt file using the below command
  uv pip install -r requirements.txt
```

```
To start the fastapi use the following command which is using uv
   uvicorn test:app --reload
```
